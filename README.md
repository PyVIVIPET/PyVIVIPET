
# PyVIVIPET

PyVIVIPET is a Python volumetric imaging viewer for PET/CT.

The tool is an open-source volumetric visualization tool for PET/CT imaging. It allows interactive exploration of PET, CT, or fused PETâ€“CT overlays, using standard medical imaging formats such as `.nii` (NIfTI).

## How to start

First clone the repository:
```
git clone https://github.com/PyVIVIPET/PyVIVIPET
```
Create and activate a virtual environment:
```
python3 -m venv venv
source ./venv/bin/activate
```
Install dependencies:

```
pip install -r requirements.txt
```
